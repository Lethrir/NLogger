Welcome to NLogger.

NLogger is a simple logging framework designed to be used in any .NET project that requires logging, including MVC sites, Windows Services and client applications.

NLogger supports output to:

  • Text Files
  • Windows Event Log

Features:

  • App.config / Web.config configuration
  • Rolling text file logger with custom file size limit and number of files limit
  • Configure severity of events to log, log diagnostics when you need but keep your files smaller when you don't
